Format du fichier CSV

Colonnes obligatoires :

  • id - Identifiant unique de l'étudiant (optionnel, sera généré automatiquement si vide)
  • matricule - Numéro matricule de l'étudiant (obligatoire)
  • first_name - Prénom de l'étudiant (obligatoire)
  • last_name - Nom de famille de l'étudiant (obligatoire)
  • email - Adresse email de l'étudiant (obligatoire)
  • password - Mot de passe de l'étudiant (obligatoire)

Configuration automatique

Les entités liées (promotion, établissement, ville, groupe, option) seront automatiquement appliquées depuis votre configuration d'importation. Vous n'avez pas besoin de les inclure dans le CSV.

Conseil : Configurez d'abord vos valeurs par défaut pour simplifier votre fichier CSV.

Téléchargement : Le modèle CSV s'adapte automatiquement à votre configuration.
