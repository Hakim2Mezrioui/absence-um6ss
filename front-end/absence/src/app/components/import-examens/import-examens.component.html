Instructions d'import

Format CSV attendu :

Le fichier doit contenir les colonnes suivantes :

  • title : Titre de l'examen (obligatoire)
  • date : Date de l'examen au format YYYY-MM-DD ou DD/MM/YYYY
  • heure_debut : Heure de début au format HH:MM, HHhMM, ou HH.MM
  • heure_fin : Heure de fin au format HH:MM, HHhMM, ou HH.MM

Note : Les autres informations (établissement, promotion, salle, etc.) seront appliquées à tous les examens importés selon la configuration ci-dessus.
