Test d'importation de fichiers Excel

Cette page permet de tester la lecture de fichiers Excel modifiés pour identifier les problèmes d'importation.

1. Test de lecture de fichier Excel

2. Test de création de modèle

3. Instructions de test

  1. Cliquez sur "Créer un modèle de test" pour télécharger un fichier Excel
  2. Ouvrez le fichier téléchargé dans Excel ou LibreOffice
  3. Modifiez quelques cellules (ajoutez des étudiants, modifiez des noms, etc.)
  4. Sauvegardez le fichier modifié
  5. Utilisez "Analyser le fichier" pour vérifier que les modifications sont détectées
  6. Vérifiez que toutes les données modifiées sont correctement lues
