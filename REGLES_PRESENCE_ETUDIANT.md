## 1. RÈGLES OBLIGATOIRES

### VOUS DEVEZ :

| Action | Description |
|--------|-------------|
| **Marquer votre présence** | À chaque cours et examen |
| **Utiliser le device Biostar** | De la salle où vous vous trouvez |
| **Pour les cours** | Marquer **avant le début** ET **après la fin** |
| **Pour les examens** | Marquer **avant le début** uniquement |
| **Arriver à l'heure** | Pour éviter les retards |
| **Vérifier** | Que votre présence est bien enregistrée |

---

### VOUS NE DEVEZ PAS :

| Interdiction | Conséquence |
|-------------|-------------|
| Utiliser un device d'une autre salle | Absence non justifiée |
| Oublier de marquer avant ou après le cours | Absence non justifiée |
| Marquer en retard sans justification | Retard enregistré |
| Quitter sans marquer votre présence | Absence non justifiée |
| Tenter de contourner le système | Sanctions disciplinaires |

---

## 2. MARQUAGE DE PRÉSENCE POUR LES COURS

### AVANT LE COURS

> **Action Requise :** Marquer votre présence **AVANT l'heure de début** du cours

| Étape | Description |
|-------|-------------|
| 1 | Arrivez dans la salle **5 à 10 minutes avant** le début |
| 2 | Repérez le **device Biostar** de la salle |
| 3 | Marquez votre présence sur le device |
| 4 | Vérifiez que votre présence est enregistrée |

---

### APRÈS LE COURS

> **Action Requise :** Marquer votre présence **APRÈS la fin** du cours

| Étape | Description |
|-------|-------------|
| 1 | À la fin du cours, restez dans la salle |
| 2 | Utilisez le **même device Biostar** de la salle |
| 3 | Marquez votre sortie sur le device |
| 4 | Vérifiez que votre sortie est enregistrée |

---

### RÈGLE IMPORTANTE

<div align="center">

**ATTENTION : Les deux marquages (AVANT et APRÈS) sont OBLIGATOIRES**

**Sans les deux marquages, vous serez considéré comme ABSENT**

</div>

## 3. MARQUAGE DE PRÉSENCE POUR LES EXAMENS

### PROCÉDURE POUR LES EXAMENS

| Étape | Description |
|-------|-------------|
| **Avant le début** | Marquez votre présence **OBLIGATOIREMENT** |
| **Device** | Utilisez le device Biostar de la salle d'examen |
| **Un seul marquage** | Suffisant pour les examens (pas de marquage de sortie) |

---

### ATTENTION EXAMENS

<div align="center">

**Tout retard à un examen peut entraîner une absence non justifiée**

**Arrivez impérativement avant l'heure de début**

</div>

## 4. CONSÉQUENCES DU NON-RESPECT

### RETARD

| Conséquence | Impact |
|-------------|-------|
| Enregistrement | Dans votre dossier personnel |
| Note de présence | Peut être affectée |
| Assiduité | Impact négatif |

---

### ABSENCE

Une absence est enregistrée comme **NON JUSTIFIÉE** si :

- Vous n'avez pas marqué votre présence **avant le cours**
- Vous n'avez pas marqué votre présence **après le cours**
- Vous avez utilisé un device d'une **autre salle**
- Vous avez quitté la salle **sans marquer votre sortie**

#### Impact de l'Absence :

| Conséquence | Description |
|-------------|-------------|
| Dossier | Enregistrée dans votre dossier |
| Évaluation | Répercussions sur votre évaluation |
| Assiduité | Affecte votre note d'assiduité |
| Sanctions | Peut entraîner des mesures disciplinaires |

## 6. VÉRIFICATION

### COMMENT VÉRIFIER VOTRE PRÉSENCE

| Action | Fréquence |
|--------|-----------|
| Consulter votre historique | **Régulièrement** |
| Vérifier les enregistrements | **Après chaque cours** |
| Contacter l'administration | **En cas de doute** |

---

### POINTS À VÉRIFIER

- Toutes vos présences (avant et après) sont enregistrées
- Aucune erreur dans votre historique
- Les horaires correspondent à vos marquages
- Aucune absence non justifiée
